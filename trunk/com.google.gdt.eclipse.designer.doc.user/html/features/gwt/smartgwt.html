Using SmartGWT

SmartGWT is a GWT based framework that allows you to not only utilize its comprehensive widget library for your application UI, but also tie these widgets in with your server-side for data management. Smart GWT is based on the powerful and mature SmartClient library. You can download the latest SmartGWT update here.

To configure your project to use SmartGWT, you can either follow the instructions on the SmartGWT web site, or you can select your GWT module file and right-click to select the Google Web Toolkit > Configure for using SmartGWT command. This will add smartgwt.jar and smartgwt-skins.jar to your classpath and module file.
