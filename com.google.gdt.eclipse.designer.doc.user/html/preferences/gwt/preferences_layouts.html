Preferences - GWT > Layouts

This page is used to control various GWT > Layouts options.

Create variable for Layout using pattern

This preference determines what pattern will be used to generate layout variables. The following styles are supported:

  • ${layoutAcronym}_${containerName} -- gl_panel
  • ${layoutAcronym}${containerName-cap} -- glPanel
  • ${containerName}${layoutClassName} -- panelGridLayout
  • ${dataDefaultName} -- gridLayout_1

Create variable for LayoutData using pattern

This preference determines what pattern will be used to generate layout data variables. The following styles are supported:

  • ${dataAcronym}_${controlName} -- gd_helloWorldButton
  • ${dataAcronym}${controlName-cap} -- gdHelloWorldButton
  • ${controlName}${dataClassName} -- helloWorldGridData
  • ${dataDefaultName} -- gridData_1
