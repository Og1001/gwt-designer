GWT FlowPanel

A FlowPanel arranges components in a directional flow, much like lines of text in a paragraph. FlowPanels are typically used to arrange buttons in a panel. It arranges buttons horizontally until no more buttons fit on the same line.
  • Main Features
  • Graphical Feedback

Main Features

  • Select FlowPanel from the Panels palette

  • Graphical feedback is provided for all drop and move interactions

 

Graphical Feedback

  • When moving a widget or adding a new widget, the current drop point is highlighted in red
